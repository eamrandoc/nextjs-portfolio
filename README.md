# 🚀 My Next.js Portfolio

🌐 Live Site: https://my-nextjs-portfolio-theta.vercel.app

A modern, responsive developer portfolio built with **Next.js**, showcasing my projects, skills, and experience in web development.

---

## ✨ Features

- ⚡ Fast and optimized with Next.js App Router
- 🎨 Modern UI with Tailwind CSS + shadcn/ui
- 🧩 Project showcase with dynamic routing
- 📱 Fully responsive design (mobile-first)
- 🌙 Smooth hover animations and transitions
- 📂 Organized project details pages
- 🔗 External links to GitHub and live demos

---

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui
- **Icons:** Lucide React, React Icons
- **Deployment:** Vercel
